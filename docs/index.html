Why Acute Health Effects of Air Pollution Could be Inflated

Hi and welcome!

This website gathers code and additional material for the paper Why Acute Health Effects of Air Pollution Could be Inflated by Vincent Bagilet and Léo Zabrocki.

The website is still under construction. It is structured as follows:

  • A PDF version of the current version of the paper is accessible through the “Paper” tab.
  • A plain language summary is available in the “Summary” tab.
  • The “Intuition” tab describes, though a simple example, the main mechanism that may lead to overestimating acute health effects of air pollution. This example is built on a simulation whose code is presented in this tab.
  • The code used to gather and analyze the literature can be accessed through the “Literature review” tab.
  • The “Simulations” tab describes the data wrangling process, a brief exploratory data analysis, the simulation choices that we made, all the code used for these simulations and resulting graphics that users can explore through a Shiny app.
  • All the R code for this project is available in its GitHub directory.
